# Notebooks

Some of my data exploration with Jupyter Notebooks.

## Getting Started

Setting up the conda environment:

```shell
位 conda env create -f environment.yml
```
Activate the conda environment:

```shell
位 conda activate notebooks
```

Run Jupyter Lab:

```
位 jupyter lab
```

You can add new packages to the `./environment.yml` file. You'll then need to
update the conda environment:

```shell
位 conda env update -f environment.yml
```
